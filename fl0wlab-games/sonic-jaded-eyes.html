Sonic: Jaded Eyes (2023 RPG Map Layout Demo)

Release Date: 5/14/2023

The first (and so far the only) demonstration of the (on-hiatus) Sonic: Jaded Eyes RPG game.

Originally developed for Crystal Vortex team developers to experience how the intended game would feel.

(Wow! a secret game! You shouldn't be here.)

Controls: Enter key to start/progress and the arrow keys to move.
