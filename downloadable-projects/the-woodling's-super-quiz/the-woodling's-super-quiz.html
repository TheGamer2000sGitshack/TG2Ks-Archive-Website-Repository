The Woodling's Super Quiz (DEMO)

Lead/Developer(s): TheGamer2000, Starg33ker

Project Description

You’ve found yourself in the Woodling’s lair! Now, it’s asking you to answer some very difficult questions about information technology! Will you answer them correctly and escape, or will you perish?

(Questions are based on CompTIA A+ Core 2 objectives.)

This project is only compatible with Windows CMD and Powershell.
