SONIC 1 GITHUB MADNESS 3

Lead/Developer(s): PikaNoob, SSRG community

Project Description

The third entry in a series of Sonic 1 April Fools' ROM hacks. The entirety of the hack barely makes any sense, but it is very much playable!

Initially created in 11 days by a large group of random Sonic ROM hackers.

(REV01 is a bugfix and DLC released a month after the initial release.)

You must have a Sega Genesis/Mega Drive console or emulator to play this project.
