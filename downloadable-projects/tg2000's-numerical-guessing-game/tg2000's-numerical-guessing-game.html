TG2000s Numerical Guessing Game

Lead/Developer(s): TheGamer2000, KingKong

Project Description

A simple game where you guess the number! Play solo in 4 difficulty modes, with a friend in 2-Player mode, or race against the time in Time Attack mode!

This version of the game is compatible with Android devices or emulators only.
